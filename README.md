# TheoChem Lab Documentation

This repository contains the public documentation website for TheoChem Lab.

Website:

https://weikii.github.io/TheoChemLab/

TheoChem Lab is currently under early internal testing. Source code, installers, and update packages are not published in this repository.

## Local Preview

```powershell
python -m pip install -r requirements.txt
python -m mkdocs serve
```

## Deployment

The site is deployed with GitHub Pages using the workflow in `.github/workflows/deploy.yml`.
