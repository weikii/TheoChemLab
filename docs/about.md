## Project Status

TheoChem Lab is currently in early internal testing. The documentation site is public, but installers and update packages are distributed only to authorized users.

The source code is not currently published.

## Scope

The program is intended for research workflows involving molecular structures, molecular crystals, intermolecular packing, charge-transfer pairs, Gaussian-related analysis, electronic coupling, anisotropic mobility, and related visualization tasks.

Scientific results produced with TheoChem Lab should be checked by the user before publication or reporting.
