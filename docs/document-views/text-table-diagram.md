# Text, Table, and Diagram Views

TheoChem Lab includes lightweight document views for non-structure information generated during a project.

## Text Document

Text documents are used for notes, logs, copied output, and plain-text results.

The text view context menu provides:

- **Copy**
- **Select All**
- **Clear**
- **Lock**
- **Save**

Use locking when the text should be preserved while reading or copying from it.

## Table Document

Table documents are used for tabular data. The table view supports copying selected table content to the clipboard.

Some analysis tools also export CSV files directly. Use table documents when you want the result to remain part of the project rather than only as an external file.

## Diagram Document

Diagram documents are used for plotted data and simple chart-style results.

The diagram view provides:

- A coordinate readout in the status bar
- **Data > Save as...** for exporting plot data
- Image export where supported by the plot tool

Use diagram documents for quick inspection and saved analysis summaries.

## Relationship to Analysis Modules

Many analysis modules create text, table, diagram, or external result files. Keep these documents in the same project folder as the source structure so the calculation path remains understandable.
