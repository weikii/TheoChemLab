# Text Document (`.txd`)

The `.txd` file is the native TheoChem Lab text document. It stores project notes, copied output, logs, generated text reports, and plain text results.

## What It Saves and Reads

| Content | Saved and read by `.txd` |
| --- | --- |
| Text content | Plain text stored in the document. |
| Document identity | File name, project location, and document type used by Project View. |
| Viewer binding | Reopens in the text viewer as a project document. |

Use `.txd` when the text should remain part of the TheoChem Lab project rather than only exist as an external text file.

## Typical Uses

- Keeping notes beside a structure
- Saving copied output from calculations or parsers
- Storing short reports generated from analysis tools
- Recording manual comments during inspection

The related view is described in [Text, Table, and Diagram Views](../document-views/text-table-diagram.md).
