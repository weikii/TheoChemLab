# TheoChem Lab

TheoChem Lab is a Windows application for molecular visualization, crystal structure inspection, and computational chemistry workflows used in molecular materials research.

!!! note "Early internal testing"
    TheoChem Lab is currently used for internal research and development. Public downloads are not available from this documentation site.

## Intended Users

TheoChem Lab is intended for researchers and students working with molecular crystals, organic semiconductors, intermolecular packing, and charge transport analysis.

Typical use cases include:

- Loading molecular and crystal structures for visual inspection
- Checking intermolecular contacts, molecular pairs, and local packing environments
- Preparing or reviewing Gaussian-related structure files
- Combining electronic coupling, reorganization energy, and packing geometry in charge-transport analysis
- Exporting plots and tabulated data for research notes or reports
