# TheoChem Lab

TheoChem Lab is a Windows application for molecular visualization, crystal structure inspection, and research workflow support in computational chemistry and molecular materials.

!!! note "Early internal testing"
    TheoChem Lab is currently used for internal research and development. Public downloads are not available from this documentation site.

## What It Provides

- Molecular and crystal structure visualization
- Project-based organization for calculation files and structures
- Crystal tools for packing, wrapping, and cell inspection
- Gaussian-oriented workflows for electronic coupling analysis
- Anisotropic mobility visualization and reporting
- Online update support for authorized internal builds

## Intended Users

TheoChem Lab is intended for researchers and students working with molecular structures, organic semiconductors, crystal packing, and charge transport analysis.

## Availability

Installers and update packages are distributed only to authorized users during the early testing stage. Source code is not published.
