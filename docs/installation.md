# Installation

TheoChem Lab is currently distributed internally.

## Requirements

- Windows 10 or Windows 11
- 64-bit operating system
- OpenGL-capable graphics environment
- Network access for online updates, if enabled

## Internal Installer

Authorized users receive an MSI installer or an internal update package from the project maintainer.

The release repository is private during early testing. Update access requires authorization.

## First Launch

After installation, start TheoChem Lab from the Start menu or desktop shortcut. If Windows asks for permission during updates, confirm the prompt to allow the updater to replace application files.
