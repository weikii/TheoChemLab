# Anisotropic Mobility

The anisotropic mobility module calculates and visualizes direction-dependent charge carrier mobility from molecular packing information and hopping parameters.

## Basic Workflow

1. Select a center molecule.
2. Add neighboring molecular pairs.
3. Enter reorganization energy and electronic coupling values.
4. Define the reference plane.
5. Generate directional mobility plots and export results.

## Theory

The workflow is based on charge hopping models commonly used for organic semiconductor crystals. The required parameters include electronic coupling, reorganization energy, molecular-pair geometry, and temperature.
