# Crystal Tools

Crystal tools help inspect, repair, and expand periodic molecular structures.

## Typical Tasks

- Make molecules whole across periodic boundaries
- Inspect unit cell geometry
- Wrap or normalize fractional coordinates
- Grow the displayed crystal environment around selected molecules
- Control whether cell axes and labels are shown

## Notes

Crystal operations should preserve the underlying crystal model unless the command explicitly edits the structure. Display-growth commands should only affect the visible environment.
