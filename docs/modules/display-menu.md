# Display and Style Menus

The Display menu controls visual appearance rather than changing chemical data.

## Display Menu Items

| Menu item | Function |
| --- | --- |
| **Style** | Open the style editor for the current document. |
| **Style Manager** | Manage built-in and user style presets. |
| **Element Colors** | Edit element-color definitions. |

## Style Editor Areas

The style editor includes:

- Atom and bond drawing modes
- Element and custom colors
- Atom radius, CPK scale, bond radius, line width
- Bond order and aromatic display style
- Render lights, materials, fake-light, and contour parameters
- Camera projection, FOV, background, and axis indicator
- Crystal display mode, draw matrix, lattice style, and anisotropic style
- Vector, surface, plane, distance/angle, number, box, fragment, and selection style sections

## Style Manager

Use Style Manager to:

- Apply a style to the current document.
- Apply a style to all documents.
- Set a style as the default.
- Create, copy, edit, rename, delete, open, or save `.wst` style files.

## Practical Use

Use display styles to make figures consistent across a project. Avoid changing structural data just to make an object visually clearer; use style settings instead.
