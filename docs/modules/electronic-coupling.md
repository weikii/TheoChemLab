# Electronic Coupling

The electronic coupling workflow supports analysis of charge transport parameters from molecular pairs and Gaussian-related outputs.

## Workflow Outline

1. Prepare molecular pair structures.
2. Run the required quantum chemistry calculations.
3. Load the output files into TheoChem Lab.
4. Extract electronic coupling values.
5. Use the values in mobility or hopping-rate analysis.

## Status

This module is under active development and documentation will be expanded with validated examples.
