# File, View, and Edit Menus

This page documents the main application-level menus before specialized build or tool windows are opened.

## File

| Menu item | Function |
| --- | --- |
| **Open...** | Open a supported file or project document. |
| **Save As...** | Save the active document to a selected path or format. |
| **Application Setting** | Open application-wide settings. |

Supported structure formats include `.wsd`, `.mol`, `.mol2`, `.gjf`, `.sdf`, `.xsd`, `.xyz`, `.arc`, `.cif`, `.pdb`, `.gro`, and `.inp`.

## View

| Menu item | Function |
| --- | --- |
| **Project View** | Show the project tree and project document navigator. |
| **Properties View** | Show editable properties for the selected object, document, or crystal. |
| **Output** | Show logs, messages, clickable viewer links, and calculation output. |
| **Job View** | Show background or external calculation jobs. |
| **Align Camera to Axis** | Align the camera to crystal axes \(a\), \(b\), or \(c\) when available. |

## Edit

| Menu item | Function |
| --- | --- |
| **Undo / Redo** | Navigate supported document edit history. |
| **Selection** | Open the selection dialog for atoms, bonds, molecules, and other objects. |
| **Atoms / Bonds** | Open edit actions related to atoms or bonds. |
| **Generate bonds** | Generate bonds from current structure geometry. |
| **Short interaction** | Generate or inspect weak/short contacts. |
| **Fragments** | Edit or select molecular fragments. |
| **Groups** | Edit object groups. |
| **Properties** | Edit properties of the current selection. |
| **Preferences** | Open user preferences. |

!!! tip
    Selection and fragment commands are often the first step before using pair, mobility, or surface-alignment tools.

For a fuller description of the project file model, supported input formats, and export behavior, see [Files and Projects](../files/index.md).
