# Gaussian Tools

Gaussian tools help organize and analyze Gaussian input and output files.

## Common Uses

- Inspect Gaussian output structures
- Extract data for downstream analysis
- Prepare electronic coupling workflows
- Connect calculation results with project documents

## Input and Output Files

TheoChem Lab can be used as a visual companion for Gaussian-oriented workflows. Typical files include prepared structures, optimized geometries, and output files that contain values needed by later analysis modules.

## Downstream Analysis

Gaussian-derived structures and values are commonly used by:

- Electronic coupling workflows for molecular pairs
- Reorganization-energy or hopping-rate calculations
- Mobility analysis that combines molecular-pair geometry with electronic parameters

## Practical Notes

Keep Gaussian outputs together with the project document when possible. This makes it easier to trace which structure, molecular pair, and extracted value were used in a later plot or table.

More detailed Gaussian workflows will be added as the internal interface stabilizes.
