# Modules

TheoChem Lab contains several modules for molecular visualization and computational chemistry workflows.

Current documentation focuses on the modules used most often during internal testing.

## Crystal Tools

Crystal tools inspect and repair periodic molecular structures. They are used before pair-based analysis when molecules cross unit-cell boundaries or when a local environment needs to be displayed around a selected structure.

## Gaussian Tools

Gaussian tools organize input/output inspection and provide values for downstream workflows such as electronic coupling and mobility analysis.

## Electronic Coupling

The electronic coupling workflow is used with molecular pairs and Gaussian-derived data to obtain charge-transfer parameters for hopping-rate analysis.

## Anisotropic Mobility

The anisotropic mobility module combines packing geometry, electronic coupling, reorganization energy, and temperature to visualize direction-dependent mobility in a selected crystal plane.
