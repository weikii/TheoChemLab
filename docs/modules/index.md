## Menu Map

| Menu | What it contains |
| --- | --- |
| **File** | Open, save, save as, application settings |
| **View** | Project View, Properties View, Output, Job View, camera-axis alignment |
| **Edit** | Undo, redo, selection, atoms, bonds, fragments, groups, properties, preferences |
| **Display** | Style, Style Manager, element colors |
| **Build** | Atoms, bonds, faces, molecule, crystal, symmetry, surface, frame sequence |
| **Tools** | Gaussian quick tools, Multiwfn/VMD visualization, ESP, anisotropic mobility, pair manager, surface interaction, loader, PXRD |
| **Modules** | Gaussian calculation/analysis, MD simulation, Gromacs, MISC calculation, Multiwfn calculation, CP2K, MolPack, Hirshfeld surface |
| **Window** | Close all document windows |
| **Help** | Online documents, update check, About |
