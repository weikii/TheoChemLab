# MolPack

MolPack provides packing and scan tools for assembling molecular arrangements.

## Pack Page

The **Pack** page is intended for setting up packed molecular systems from selected or imported molecular components.

## Scan Page

The **Scan** page includes axis buttons:

- X
- Y
- Z

Use scan controls to explore directional packing changes or simple coordinate scans.

## Relationship to Gromacs

MolPack is useful for structure generation and exploratory packing. Gromacs generation is more oriented toward simulation input assembly with molecule counts, residue names, and topology-related files.

## Recommended Use

- Use MolPack for fast packing experiments.
- Inspect results visually in the 3D Structure Viewer.
- Use Gromacs generation when preparing MD-oriented systems.
