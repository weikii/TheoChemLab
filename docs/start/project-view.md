# Project View

Project View is the file and document navigator for a TheoChem Lab project.

## Purpose

Use Project View to keep related files together:

- Structure documents
- Imported input files
- Gaussian outputs
- Generated analysis files
- Notes, tables, and diagrams
- Folders used to organize a research task

## Opening Items

Open items from the tree to show them in the document area. Structure documents open in the 3D Structure Viewer. Text, table, and diagram documents open in their own document views.

## Creating Documents

Use the main **New** button or Project View commands to create project documents:

- **3D Structure Document**
- **Text Document**
- **Table Document**
- **Diagram Document**

The default 3D structure document name is **3D Structure Document.wsd**.

## File Names and Tooltips

Project View displays file and document names in the tree. If a name is too long to fit in the current panel width, the tooltip can show the full name. Items with extra action icons reserve space so icons do not cover the visible file name.

## Project Organization

A practical project layout is:

- One folder for source structures
- One folder for Gaussian or external calculation files
- One folder for generated plots, tables, or reports
- One or more 3D structure documents for visualization and analysis states

Keep related outputs near the structure that generated them. This makes later analysis easier to trace.
