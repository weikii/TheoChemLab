# Welcome Screen

The welcome screen is the first project-level entry point in TheoChem Lab.

## Choose How to Start

The screen provides two main choices:

- **Create a new project**: create a new `.wlp` project file.
- **Open an existing project**: load an existing `.wlp` project file.

Use **Browse** to select an existing project manually. The recent-project list can also be used to reopen a project that was used before.

## Recent Project Startup

The option **Always open the most recently used project on start-up** skips manual selection and opens the latest project automatically on the next launch.

Use this only when you usually work in one active project. If you regularly switch between projects, leave it off so the welcome screen remains the normal starting point.

## Project Files

TheoChem Lab project files use the `.wlp` extension. A project is the container for related documents, imported files, generated files, and analysis outputs.

Typical project content includes:

- 3D structure documents
- Imported structure files
- Gaussian output files
- Text notes
- Tables and diagrams
- Calculation or analysis results
