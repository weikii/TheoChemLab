# Updates

TheoChem Lab supports online updates for authorized internal builds.

## Versioning

The application uses two values to determine update availability:

- Stable version, for example `0.0.1`
- Build time, for example `20260625.12.00`

An update is available when the remote stable version is newer, or when the stable version is the same but the remote build time is newer.

## Access

During early testing, update packages are stored in a private release repository and require authorization.
