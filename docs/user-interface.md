# User Interface

TheoChem Lab is organized around a main structure viewer and several dockable utility views.

## Main Viewer

The main viewer displays molecules, crystals, surfaces, orbitals, and annotation objects.

## Project View

The project view lists project files and generated results. It is intended for repeated navigation across structures, input files, and calculation outputs.

## Properties View

The properties view exposes editable settings for the currently selected object, document, or crystal.

## Output and Jobs

Output and job views collect calculation logs, status messages, and background workflow information.
